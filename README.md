# 📌 Project Overview

Kavya-Kanaja is a modern Android application developed to preserve, promote, and simplify Kannada literature for younger generations. The application provides an interactive and educational platform where users can explore Kannada poems, understand their meanings, learn about poets, and engage with literature through a clean and user-friendly mobile experience.

The project focuses on solving the declining interest in classical Kannada poetry among students and Gen-Z audiences by making poetry learning more engaging, accessible, and digitally interactive.

This application was developed as part of the **MindMatrix Android App Development using GenAI Internship Program**.

---

# 🎯 Problem Statement

Many students and young learners find it difficult to understand classical Kannada poetry because of:

- Complex vocabulary and traditional literary style
- Lack of accessible digital learning platforms
- Limited availability of simplified explanations (Bhavartha)
- Low engagement with conventional literature learning methods
- Absence of interactive offline educational applications for Kannada poetry

As a result, students often memorize poems for academic purposes without truly understanding their meaning or cultural significance.

Kavya-Kanaja addresses this issue by providing:

- Simplified poem explanations
- Poet information
- Interactive learning features
- Offline accessibility
- Clean and engaging reading experience

# 🚀 Key Features

## 📖 Poem of the Day
- Displays a daily featured Kannada poem
- Encourages regular engagement with literature
- Dynamic poem rendering based on date

## 📚 Poetry Library
- Collection of Kannada poems
- Easy browsing and navigation
- Search and filter functionality

## 🧠 Bhavartha (Meaning Section)
- Simplified explanation for difficult Kannada words
- Interactive word meaning popups
- Helps students understand classical literature

## 👨‍🎓 Poet Information
- Displays poet biographies
- Literary contributions and achievements
- Educational reference for learners

## 🔖 Favorites & Bookmarks
- Save favorite poems locally
- Quick access to bookmarked poems

## 🗂️ Reading History
- Tracks recently viewed poems

## 🌐 Offline Support
- Core application works without internet connection
- Local Room Database support

## 🔤 Transliteration Support
- Improves pronunciation and readability

## 🎨 Modern UI/UX
- Developed using Jetpack Compose
- Material Design 3 implementation
- Responsive layouts and smooth navigation
